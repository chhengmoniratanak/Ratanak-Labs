# Ratanak's Engineering Portfolio: Architecture, Security & Automation

Welcome to my personal engineering space. Originally built as a documentation hub for my multi-vendor enterprise network simulations, this repository has evolved into a complete portfolio of my technical progression. It houses everything from complex cloud-hosted EVE-NG lab environments to automation scripts and verified industry certifications.

## 🗂️ Repository Structure

This repository is organized into three main functional areas:

### 📁 `Projects/`
My core engineering space for building and testing multi-vendor networks. To push past physical hardware constraints, these environments are deployed using cloud-hosted EVE-NG. 
* **Topology Files:** Live EVE-NG `.unl` project files to export and inspect the control plane.
* **Configurations:** Production-ready, sanitized running configurations organized by device role.
* **Documentation:** Clear breakdowns of IP schemas, routing logic, and security zones.

### 📁 `automation script/`
Where I bridge the gap between traditional CLI management and modern network programmability. 
* Python scripts for pushing configurations.
* Automated network state verification and data parsing.
* Tools designed to reduce human error and speed up deployment times.

### 📁 `Certifications/`
A timeline of my formal skill acquisition and technical discipline. This folder contains verifications for both high-stakes vendor exams and continuous self-study courses.
* **Industry Certifications:** Cisco CCNA, Fortinet, and other major vendor credentials.
* **Skills & Courses:** Certificates of completion for specialized deep-dive courses and technical bootcamps.

## 🛠️ Tech Stack & Core Focus

* **Core Routing & Switching:** Deep understanding of the control plane, utilizing Multi-Area OSPF, BGP path manipulation, and route redistribution.
* **Next-Gen Security:** Designing zone-based policies, deploying Route-Based IPsec VPNs (VTIs), and hardening infrastructure using Fortinet and Palo Alto fabrics.
* **Overlay & Tunneling:** Scaling architectures using modern tunneling methods (GRE over IPsec, DMVPN) to connect enterprise sites securely.
* **Automation:** Leveraging Python and automation frameworks to standardize configurations and manage infrastructure at scale.
